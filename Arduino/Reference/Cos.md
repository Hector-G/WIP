# cos()

### Sintaxis

```cos(rad)```

### Descripción

Calcula el coseno de un ángulo (en radianes). El resultado estará entre -1 y 1.

### Parámetros

rad: el ángulo en radianes (_float_)

### Devuelve

El coseno del ángulo (_double_)
