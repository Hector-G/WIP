# += , -= , *= , /= , %=

### Descripción

Realiza una operación matemática sobre una variable usando otra constante o variable. No es más que una manera de abreviar de forma práctica expresiones habituales.

### Sintaxis

```
x += y;   // equivale a la expresión x = x + y;
x -= y;   // equivale a la expresión x = x - y; 
x *= y;   // equivale a la expresión x = x * y; 
x /= y;   // equivale a la expresión x = x / y; 
x %= y;  //  equivale a la expresión x = x % y; 
```

### Parámetros

**x**: cualquier tipo de variable

**y**: cualquier tipo de variable o constante

### Ejemplos

```
x = 2;
x += 4;      // x ahora contiene 6
x -= 3;      // x ahora contiene 3
x *= 10;     // x ahora contiene 30
x /= 2;      // x ahora contiene 15
x %= 5;      // x ahora contiene 0
```
