# ; (punto y coma)

Se utiliza para finalizar una instrucción.

### Ejemplo

```int a = 13;```

### Consejo

Olvidarse de un punto y coma al final de una linea tendrá como resultado un error de compilador (*Verificar*). El error puede ser obvio, y el compilador indicará que falta un punto y coma, pero a veces puede no ser tan claro. Si el programa presenta errores extraños al verificar, una de las primeras cosas que hay que comprobar es que no falte un punto y coma en algún lugar cercano a la linea donde se está indicando el error.
